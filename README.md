# 🛡️ RoshanAdsHider (RASH)

RoshanAdsHider (RASH) is a lightweight, high-performance visual element blocker for Tampermonkey. It features a modern **Liquid Glass (Glassmorphism)** UI and allows you to clean up your web experience effortlessly.

---

## ✨ Features / คุณสมบัติเด่น

### 🇬🇧 English
*   **Liquid Glass UI:** Beautifully blurred, transparent, and modern interface.
*   **Continuous Blocking:** Click "BLOCK" once and hide as many elements as you want without page refreshes.
*   **Stable & Persistent:** Fixed position (top-right) so it never gets lost, and your hidden rules are saved forever.
*   **Undo Support:** Accidentally hid something? Just press `ClearAll` to bring it back.

### 🇹🇭 ภาษาไทย
*   **อินเตอร์เฟสแบบ Liquid Glass:** สวยงาม ทันสมัย พร้อมเอฟเฟกต์กระจกเบลอโปร่งแสง
*   **เลือกปิดได้ต่อเนื่อง:** กดปุ่ม "BLOCK" เพียงครั้งเดียว แล้วเลือกจิ้มซ่อนโฆษณาได้รัวๆ โดยไม่ต้องรอโหลดหน้าใหม่
*   **เสถียรและถาวร:** ปุ่มล็อคตำแหน่งอยู่มุมขวาบน ไม่หายไปไหนแน่นอน พร้อมระบบจำกฎการซ่อนแม้ปิดเบราว์เซอร์
*   **ระบบ Undo:** ถ้าเผลอจิ้มผิด เพียงกด `ClearAll` เพื่อดึงสิ่งที่ซ่อนกลับมา

## 🚀 How to Install / วิธีติดตั้ง

1.  **Install Tampermonkey:** Install the Tampermonkey extension for your browser 
([Chrome](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo), 
[Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/), 
[Edge](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepogpeiohjachj)).
2.  **Add Script:** Click the **Raw** button on the `RoshanAdsHider.user.js` file in this repository.
3.  **Confirm:** Tampermonkey will open a new tab. Click **Install**.

### 🚀 Installation / วิธีติดตั้ง
[คลิกที่นี่เพื่อติดตั้ง R.A.S.H.](https://github.com/RoshanzeroX/R.A.S.H.-Roshan-Ads-Selective-Hider-/raw/refs/heads/main/RoshanAdsHider.user.js)

## 🛠️ How to Use / วิธีใช้งาน

1.  **Open Menu:** Click the small red pulsing circle at the top-right of any webpage.
    *(คลิกที่จุดสีแดงมุมขวาบนของหน้าเว็บเพื่อเปิดเมนู)*
2.  **Enter Block Mode:** Click the **BLOCK** button (it will turn green and say "EXIT").
    *(กดปุ่ม **BLOCK** เพื่อเข้าสู่โหมดเลือกซ่อน ปุ่มจะกลายเป็นสีเขียว)*
3.  **Hide Elements:** Hover over any ad or element you want to hide (a red dashed frame will appear) and **Click** it.
    *(นำเมาส์ไปชี้โฆษณาที่จะซ่อนจนขึ้นกรอบแดง แล้วทำการ **คลิก**)*
4.  **Finish:** Click **EXIT** to stop blocking.
    *(เมื่อเสร็จแล้วให้กด **EXIT** เพื่อออกจากโหมดเลือก)*
5.  **Reset:** Use **CLEAR ALL** to show everything again.
    *(หากต้องการยกเลิกการซ่อนทั้งหมด ให้กดปุ่ม **CLEAR ALL**)*
